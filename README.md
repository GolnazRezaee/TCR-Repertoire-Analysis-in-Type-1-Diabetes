# TCRβ Repertoire Analysis in Type 1 Diabetes
**Erasmus Research Internship — Technical University of Denmark (DTU), 2025–2026**  
Supervisor: Prof. Leon Eyrich Jessen

## Overview
Comparative analysis of TCRβ repertoire diversity and clonotype enrichment 
between Type 1 Diabetes patients (n=298) and healthy controls (n=131), using 
AIRR-compliant immune receptor sequencing data from 908 individuals.

## Key Findings
- T1D patients show significantly higher Shannon diversity than controls
  (p < 0.001, Wilcoxon rank-sum test)
- Median diversity: T1D = 0.978 vs CTRL = 0.960
- Diversity declines with age across both groups; T1D maintains higher
  diversity throughout
- Female T1D patients show consistently elevated diversity; males show
  stronger age-related decline
- 388,988 clonotypes show higher relative prevalence in T1D vs
  1,608,633 in controls
